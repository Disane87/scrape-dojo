## So funktioniert es

Statt hunderte Zeilen Puppeteer-Code zu schreiben, beschreibst du deinen Workflow deklarativ:

```jsonc
{
  "scrapes": [{
    "id": "invoice-download",
    "metadata": {
      "description": "Rechnungen automatisch herunterladen",
      "triggers": [{ "type": "cron" }]
    },
    "steps": [{
      "name": "Login und Download",
      "actions": [
        { "name": "open", "action": "navigate",
          "params": { "url": "https://portal.example.com/login" } },
        { "name": "email", "action": "type",
          "params": { "selector": "#email", "text": "{{variables.email}}", "pressEnter": false } },
        { "name": "password", "action": "type",
          "params": { "selector": "#password", "text": "{{variables.password}}", "pressEnter": true } },
        { "name": "invoices", "action": "extractAll",
          "params": { "selector": ".invoice-link", "property": "href" } },
        { "name": "downloadLoop", "action": "loop",
          "params": {
            "elementKey": "invoices",
            "actions": [
              { "name": "dl", "action": "download",
                "params": {
                  "url": "{{currentData.downloadLoop.value}}",
                  "path": "./invoices",
                  "filename": "invoice-{{currentData.downloadLoop.index}}.pdf"
                }}
            ]
          }}
      ]
    }]
  }]
}
```

Secrets sind AES-256-CBC verschluesselt, die UI zeigt den Fortschritt live per SSE, und Cron-Scheduling startet den Job automatisch.

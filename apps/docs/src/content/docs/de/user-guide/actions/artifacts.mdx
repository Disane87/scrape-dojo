Mit `artifacts` machst du deine Scrape-Ergebnisse sichtbar — als Tabelle, JSON-Viewer, Karte oder Bild, direkt im UI. Das ist mega nützlich zum Debuggen, aber auch für hübsche Dashboards und Berichte. Der Typ wird automatisch erkannt, du kannst ihn aber auch manuell setzen.

## Parameter

| Parameter | Typ | Pflicht | Beschreibung |
|-----------|-----|---------|--------------|
| `data` | `any` | Ja | Die anzuzeigenden Daten (unterstützt Handlebars-Templates) |
| `type` | `string` | Nein | Darstellungstyp: `auto`, `json`, `table`, `text`, `file`, `image`, `link`, `card` (Standard: `auto`) |
| `title` | `string` | Nein | Titel des Artifacts |
| `description` | `string` | Nein | Beschreibung unter dem Titel |
| `template` | `string` | Nein | Handlebars-Template für den `card`-Typ |
| `flush` | `boolean` | Nein | Kompakte Listen-Darstellung für Karten |

### Auto-Erkennung

Wenn `type` auf `auto` steht (Standard), wird der beste Darstellungstyp automatisch ermittelt:

| Daten | Erkannter Typ |
|-------|--------------|
| Array von Objekten | `table` |
| Objekt | `json` |
| String mit Bild-Endung (`.png`, `.jpg`, …) | `image` |
| String mit URL (`https://…`) | `link` |
| String (JSON-parsebar) | `json` |
| Sonstiger String | `text` |

## Beispiele

**Tabellen-Artifact:**

```jsonc
{
  "name": "showProducts",
  "action": "artifacts",
  "params": {
    "data": "{{previousData.products}}",
    "title": "Gefundene Produkte",
    "description": "Alle extrahierten Produktdaten"
  }
}
```

Wenn `products` ein Array von Objekten ist, wird automatisch eine Tabelle gerendert.

**JSON-Artifact:**

```jsonc
{
  "name": "showRawData",
  "action": "artifacts",
  "params": {
    "data": "{{previousData.apiResponse}}",
    "type": "json",
    "title": "API Response"
  }
}
```

**Karte mit Handlebars-Template:**

```jsonc
{
  "name": "showProductCard",
  "action": "artifacts",
  "params": {
    "data": "{{previousData.product}}",
    "type": "card",
    "title": "Produktdetails",
    "template": "<h3>{{name}}</h3><p>Preis: {{price}} €</p><p>{{description}}</p>"
  }
}
```

Das `template` wird mit den Daten aus `data` gerendert — du hast also direkten Zugriff auf die Felder des Objekts.

**Bild-Artifact:**

```jsonc
{
  "name": "showScreenshot",
  "action": "artifacts",
  "params": {
    "data": "{{previousData.screenshotPath}}",
    "type": "image",
    "title": "Screenshot der Seite"
  }
}
```

<Aside type="tip">
Nutze `flush: true` bei Karten, um eine kompakte Listen-Darstellung zu bekommen — ideal wenn du viele kleine Karten hintereinander anzeigen willst.
</Aside>

<Aside type="note">
Artifacts werden in Echtzeit ans UI gesendet. Du siehst sie sofort in den Run-Details, während der Scrape noch läuft.
</Aside>

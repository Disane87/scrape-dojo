## Welcome to Scrape Dojo

*December 30, 2025*

Welcome to the official documentation of **Scrape Dojo** - your web scraping framework!

### What is Scrape Dojo?

Scrape Dojo is a flexible and powerful framework for web scraping and automation. With JSON-based configuration files, you can define and automate complex scraping tasks without having to write new code every time.

### Main Features

- 🎯 **JSON-based Configuration**: Define your scrapes declaratively
- 🔄 **Dynamic Templates**: Use template variables for flexible workflows
- 🤖 **Automation**: Schedule and automate your web tasks
- 📊 **Data Extraction**: Extract structured data from websites
- 🛠️ **Extensible**: Easy to extend with custom actions and plugins

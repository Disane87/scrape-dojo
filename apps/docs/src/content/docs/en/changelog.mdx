<Aside type="tip">
The full changelog is automatically generated by [semantic-release](https://github.com/semantic-release/semantic-release) and is also available directly on [GitHub](https://github.com/disane87/scrape-dojo/blob/main/CHANGELOG.md).
</Aside>

The changelog shows all published versions with their associated changes.

## Commit Types

| Type | Meaning |
|------|---------|
| 🚀 feat | New feature |
| 🛠️ fix | Bug fix |
| ⚡ perf | Performance improvement |
| 📔 docs | Documentation |
| ♻️ refactor | Code refactoring |
| 🧪 test | Tests |
| 📦 build | Build system |
| 🤖 ci | CI/CD |

## Versions

You can find the current release notes on the [GitHub Releases page](https://github.com/disane87/scrape-dojo/releases).

In the application itself, you can access the changelog at any time via the user menu.

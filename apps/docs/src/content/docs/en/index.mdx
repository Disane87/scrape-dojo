## The Problem

Have you ever spent hours writing Puppeteer code just to scrape data from a single website? Or had to duplicate and modify the same code for ten different sites? 😅

**Scrape Dojo** is here to make your life easier. Imagine being able to describe your web scraping workflows in JSON - just like defining infrastructure as code!

## The Solution

Hundreds of lines of Puppeteer code become:

```json
{
  "id": "my-scrape",
  "steps": [
    {
      "name": "Login",
      "actions": [
        {"action": "navigate", "params": {"url": "https://example.com"}},
        {"action": "type", "params": {"selector": "#email", "text": "{{secrets.email}}"}},
        {"action": "type", "params": {"selector": "#password", "text": "{{secrets.password}}"}},
        {"action": "click", "params": {"selector": "button[type=submit]"}}
      ]
    },
    {
      "name": "Extract",
      "actions": [
        {
          "action": "extract",
          "params": {
            "selector": ".item",
            "extractAll": true,
            "properties": {
              "title": ".title",
              "price": ".price"
            }
          }
        }
      ]
    }
  ]
}
```

**That's it!** 🎉 Secrets are encrypted, logging is automatic, the UI shows everything in real-time!
